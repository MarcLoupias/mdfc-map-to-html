2 Installation et configuration

Comment installer Git sur macOS ?

Il existe plusieurs façons d’installer Git sous macOS.

  • La première façon consiste à installer XCode qui inclut Git (Apple Git-xy). Depuis un terminal, saisir la ligne de commande => xcode-select --install.
  • La deuxième façon consiste à utiliser Git-osx-install. Téléchargez le fichier .dmg et lancez l’installation en suivant les instructions.
  • La troisième façon consiste à utiliser Homebrew. En supposant que Homebrew soit installé, depuis un terminal saisir la ligne de commande => brew install git.
